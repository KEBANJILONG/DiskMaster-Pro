## English

### 📋 Overview

DiskMaster Pro is a professional disk management tool designed to help users efficiently manage disk space, analyze usage patterns, and optimize storage performance.

### ✨ Features

- 📊 **Disk Analysis** - Visualize disk usage with intuitive charts
- 🧹 **Smart Cleanup** - Identify and remove unnecessary files
- 🌐 **Multi-language Support** - English, 中文, and more
- ⚡ **High Performance** - Optimized for large drives
- 🔒 **Safe Operations** - Preview before any changes
- 📈 **Usage Reports** - Detailed statistics and trends

### 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/KEBANJILONG/DiskMaster-Pro.git

# Install dependencies
pip install -r requirements.txt

# Run the application
python main.py
```

### 🛠️ Installation

#### Requirements
- Python 3.8+
- Windows 10/11, macOS, or Linux

#### From Source
```bash
git clone https://github.com/KEBANJILONG/DiskMaster-Pro.git
cd DiskMaster-Pro
pip install -e .
```

## 中文

### 📋 概述

DiskMaster Pro 是一款专业的磁盘管理工具，旨在帮助用户高效管理磁盘空间、分析使用模式并优化存储性能。

### ✨ 功能特点

- 📊 **磁盘分析** - 通过直观的图表可视化磁盘使用情况
- 🧹 **智能清理** - 识别并删除不必要的文件
- 🌐 **多语言支持** - 英语、中文等
- ⚡ **高性能** - 针对大容量驱动器优化
- 🔒 **安全操作** - 任何更改前可预览
- 📈 **使用报告** - 详细的统计和趋势

### 🚀 快速开始

```bash
# 克隆仓库
git clone https://github.com/KEBANJILONG/DiskMaster-Pro.git

# 安装依赖
pip install -r requirements.txt

# 运行应用

```

### 🛠️ 安装

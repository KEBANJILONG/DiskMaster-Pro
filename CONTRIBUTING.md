# 🤝 Contributing to DiskMaster Pro

Thank you for your interest in contributing to DiskMaster Pro! We welcome contributions from the community.

## 🌐 Languages

- [English](#english)
- [中文](#中文)

---

## English

### 🚀 Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/DiskMaster-Pro.git`
3. Create a branch: `git checkout -b feature/your-feature`
4. Make your changes
5. Submit a pull request

### 📋 Guidelines

- Follow PEP 8 style guide for Python code
- Write clear commit messages
- Add tests for new features
- Update documentation as needed

### 🐛 Reporting Bugs

Please use GitHub Issues and include:
- Operating system
- Python version
- Steps to reproduce
- Expected vs actual behavior

## 中文

### 🚀 开始贡献

1. Fork 本仓库
2. 克隆你的 fork: `git clone https://github.com/YOUR_USERNAME/DiskMaster-Pro.git`
3. 创建分支: `git checkout -b feature/your-feature`
4. 进行修改
5. 提交 Pull Request

### 📋 指南

- Python 代码遵循 PEP 8 风格
- 编写清晰的提交信息
- 为新功能添加测试
- 根据需要更新文档

### 🐛 报告 Bug

请使用 GitHub Issues 并包含：
- 操作系统
- Python 版本
- 复现步骤
- 预期行为与实际行为
